Can your model think like an astrophysicist?

AstroBench is a rigorously engineered LLM benchmark built from real quantitative astrophysics. It's the only open benchmark in the space that targets reasoning, derivation, and unit-correct quantitative problem solving across cosmology, exoplanets, stellar physics, general relativity, and data analysis. No multiple choice. No trivia.

View the problem set →
Why astrophysics? 
Astrophysics problems encode real domain reasoning: algebra + physics intuition + multi-step
derivations. They’re more discriminating than coding benchmarks (where memorization helps)
and genuinely scientific (not trivia). Your PhD background in astrophysics gives us curated
problems with real reference solutions — impossible to game with web-scraping alone.
How is this different from MMLU, GSM8K, etc.? 
Those are broad-domain multiple-choice. AstroBench uses fully quantitative open-response
problems with reference-solution grading. A model can’t luck into the right answer without
showing correct reasoning, units, and numerical accuracy.
How do I submit a result? 
Run: astrobench run --provider openai --model gpt-4o --output result.json
Then open: astrobench submit https://github.com/Brighton-Lyrax/astrobench
